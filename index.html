Portfolio

I created this 1 page website using HTML, CSS3 and Bootstrap after completing the Bootstap module of a Web Development course I am currently working through. I was originally inspired by the many parallax sites where an image is placed in the background with content on top. I created a simple version for my site using an image of visually one of my favourite parts of London - Piccadilly Circus.

My main takeaway from this project has been to take my time to regularly refactor my code to ensure it is easy to read, and easy to reuse by myself or anyone else in the future.

Dice Game

A simple 2 player game of dice. I used the Math.Random() function to randomly choose the dice faces. The banner also changes to announce the winner, or to announce the game is a draw; I used setAttribute on the h1 along with if statements to achieve this.

Tindog

A dating site for Dogs! This was the result of a HTML and CSS challenge which I completed. I was provided with the images, colour scheme and text, and I needed to style it using Bootstrap first before CSS. When I completed this challenge I was inspired to re-create my own webpage

CSS site

Another challenge accepted to create a simple one page website. The aim was to prctice using the Float and Position elements, and choosing colours.

Drink Water App

An app to track the users Water Intake. Admittedly this isnt my own creation however it gave me the opportunity to practice shape making using CSS, and to see JavaScript in action. I do plan to add to this project to make it more of my own

Alarm Clock

Not my creation. It was interesting to see how straight forward it was to create the clock using JavaScript. My goal is to add to this clock to turn it into a working alarm clock

QR Code Challenge

A simple challenge I found on the Frontend Mentor site. Although it looks easy it was a great way to practice my CSS Skills.
